1. Download docker from : https://docs.docker.com/desktop/install/windows-install/
2. Install docker
3. Open powershell and run the following command to update wsl: wsl --update

## Install WSL2
1. Open powershell and run the following command to install wsl2: wsl --install
2. Restart your computer

## Install Ubuntu
1. Open powershell and run the following command to install ubuntu: wsl --install -d Ubuntu
2. Restart your computer

## Install Docker in WSL2
1. Open Ubuntu and run the following commands:
```
sudo apt-get update
sudo apt-get install docker.io
```
